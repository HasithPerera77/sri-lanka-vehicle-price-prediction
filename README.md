#### **Sri Lanka Vehicle Price Prediction**



**#Project Overview**



This project builds a Machine Learning model to predict vehicle prices based on vehicle specifications such as engine size, horsepower, and weight.



**#Dataset**



Vehicle dataset containing:



\* Price

\* Engine Size

\* Horsepower

\* Weight

\* Manufacturer



**#Tools Used**



Python

Pandas

Matplotlib

Seaborn

Scikit-learn

Jupyter Notebook



**#Machine Learning Model**



Linear Regression model used for price prediction.



**#Analysis Performed**



\* Data cleaning

\* Feature selection

\* Price vs horsepower visualization

\* Model training and prediction

\* Error evaluation



**#How to Run**



Install dependencies



pip install pandas matplotlib seaborn scikit-learn



Open



analysis.ipynb



Run all cells.
